Embedding

Get the embedding as a CSV file. The two dimensional embedding is shown as an image on the backend, and this file also contains the n-dimensional embedding (n specified when experiment created).

embedding.csv
This file includes coordinates for each embedding. These coordinates are associated with an stimulus label.

i.e.
stimulus_name,x_cord,y_cord,x_1,x_2,x_3
vonn.png,0.5,-0.5,0,1,2
bode.png,-1.0,-0.5,0,2,1
ligety.png,-0.5,-0.75,1,2,0
{x,y}_cord are for the two dimensional embedding and x_i are for the n-dimensional embedding.
